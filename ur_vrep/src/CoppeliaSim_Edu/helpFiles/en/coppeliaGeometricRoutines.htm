Coppelia geometric routines (auxiliary API)

This collection of functions allows to perform the same geometric calculations as you are able to do from within CoppeliaSim, i.e. collision detection, minimum distance calculation, and proximity sensor simulation on meshes, oc-trees and point clouds.

Refer to the geom.h file in the Coppelia geometric routines repository for API details (for now).

The Coppelia geometric routines source code is not directly part of CoppeliaSim, and carries separate licensing conditions. Refer to the source code for details.
